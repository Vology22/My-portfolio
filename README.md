## 🛠 Проекты

### 🤖 Bot-auto-reg
**Telegram-бот для автоматизации рассылок и лидогенерации.**
* **Функционал:** Массовая рассылка сообщений по Telegram-группам для привлечения целевой аудитории.
* **Особенности:** * Встроенный автоответчик.
    * Автоматическая квалификация пользователя (проверка готовности к работе) перед переводом диалога на оператора.
    * Тонкая настройка параметров.
    * **Mobile-friendly:** Полная возможность управления ботом прямо со смартфона.

### 💬 BotDialog
**Инструмент для симуляции активности в Telegram-сообществах.**
* **Назначение:** Создание видимости живого общения в группах путем планирования длительных диалогов между несколькими аккаунтами.
* **Технологии:** Интеграция с нейросетями для генерации реалистичных и контекстных ответов.

### 🏋️ Bot-GYM
**Классический информационный бот-справочник.**
* **Суть:** Демонстрация реализации простого и понятного интерфейса для предоставления структурированной информации пользователю. Ничего лишнего — только эффективная доставка контента.

### 🔍 ParserHouzz
**Специализированный парсер для зарубежного рынка.**
* **Задача:** Сбор данных с немецкого портала *Houzz*.
* **Результат:** Автоматизированный поиск архитекторов и дизайнеров, извлечение их контактных данных и последующая сортировка.

### 🖼 ParserUserVK-GUI
**Парсер пользователей ВКонтакте с графическим интерфейсом.**
* **Функционал:** Сбор целевой аудитории из различных сообществ VK.
* **Особенности:** Наличие удобного GUI (графического интерфейса) для комфортного взаимодействия с программой без использования терминала.
